# GoldenCheck Examples

| Script | Description |
|--------|-------------|
| `scan_basic.py` | Scan a CSV file and print all findings |
| `validate_rules.py` | Validate data against a `goldencheck.yml` config |
| `domain_pack.py` | Scan with the healthcare domain pack for clinical data types |

## Prerequisites

```bash
pip install goldencheck
```
